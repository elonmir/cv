# CV Page Design

A modern, printable CV/resume page built with Next.js App Router, TypeScript, and Tailwind CSS.

## Highlights

- Bilingual CV data (`English` and `German`) from JSON files
- Reusable CV-focused components (header, timeline, skills, testimonials, toolbar)
- Theme toggle and print action support
- Contact QR code section
- JSON-driven SEO metadata (`meta.title`, `meta.description`)
- Locale-aware Open Graph image endpoint at `/og?lang=en|de`
- Optimized static assets in `public/`

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- pnpm

## Content Editing

Update CV content and metadata in:

- `data/cv.en.json`
- `data/cv.de.json`

Keep both locale files aligned to the shared shape in `lib/cv-types.ts`.

Top-level JSON sections:

- `meta` - SEO title and description used by `app/layout.tsx`
- `ui` - section labels and UI copy
- `basics`, `experience`, `education`, `volunteering`, `skills`, `languages`, `hobbies`, `testimonials`

## Open Graph and Social Metadata

- Global metadata is configured in `app/layout.tsx`
- Social preview image is generated by `app/og/route.tsx`
- Default OG/Twitter image URL uses English metadata: `/og?lang=en`
- German variant is available at: `/og?lang=de`
- Profile image is read from `basics.photo` in the selected locale JSON

## Key Files

- `app/page.tsx` - main entry page
- `app/layout.tsx` - global metadata and icons
- `app/og/route.tsx` - locale-aware OG image generator
- `components/cv/cv-app.tsx` - CV composition container
- `components/cv/cv-header.tsx` - header/sticky behavior
- `app/globals.css` - global styling
